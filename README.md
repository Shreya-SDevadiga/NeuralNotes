# NeuralNotes 🧠

An AI-powered note-taking app for students. Upload lectures, generate notes, create flashcards, and visualize concepts.

## Features
- 📝 AI-generated notes from lectures
- 🃏 Flashcard generator
- 🗺️ Concept map visualization
- 📊 Analytics dashboard
- 🎙️ Live lecture transcription
- 📤 Export notes to PDF

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **AI:** Groq API, Anthropic Claude

## Getting Started

### 1. Clone the repo
git clone https://github.com/Shreya-SDevadiga/NeuralNotes.git

### 2. Setup Server
cd server
npm install
node index.js

### 3. Setup Client
cd client
npm install
npm start
